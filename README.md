# Python Tip Calculator

## 📌 Project Overview
This is a simple Tip Calculator project built using Python.

The program calculates:
- Tip amount
- Total bill
- Bill per person (if split)

This project was created to practice Python arithmetic and logical operators.

---

## 🚀 Features
- Takes user input for bill amount
- Calculates tip based on percentage
- Splits bill among multiple people
- Basic input validation
- Clean formatted output

---

## 🧠 Concepts Used
- Variables
- Data Types (int, float)
- Arithmetic Operators (+, -, *, /)
- Logical Operators (and)
- Conditional Statements (if-else)

---

## ▶️ How to Run

1. Make sure Python is installed.
2. Download the file `tip_calculator.py`
3. Open terminal in the file location.
4. Run:

   python tip_calculator.py
